Work

Apploi Mobile 2.0 — Revamp mobile visual design and user experience for Apploi, a job search app, to pair with the newly designed and repurposed web consumer application. This project's goal is to increase engagement, repeat usage, and focuses on Search as a core action and granular filtering for enhanced personalization. View project details.

Apploi Web 2.0 — Preliminary research, design sprints, information architecture, prototyping, and user testing to rethink Apploi's online job application process. The goal is to make applying for a job as easy, effective, and enjoyable as possible and dramatically enhance leads, click-throughs, and success rate of each application and qualitatively improve the user's entire experience. View project details. | View finished product. Collaborators: C. Cassidy, G. Narich, T. Gebauer, M. Raton

Apploi Enterprise 2.0 — Information architecture, content strategy, and prototyping for Apploi's enterprise dashboard used by hiring managers. View project details. | View finished product. Collaborators: T. Gebauer, G. Narich, M. Raton, S. Hanson
